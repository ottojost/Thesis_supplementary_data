# DNA Templates and Sequences

This repository contains the DNA sequences used throughout the experiments detailed in the thesis. It includes the original plasmid sequence, the primers used for the successful PCR amplification, and the resulting amplicon that served as templates for the RPA reactions. Within the sequence files, uppercase letters indicate the core gBlock sequence representing the actual target template (HINTW gene). Lowercase letters denote the flanking vector DNA required to complete the plasmid, which is not part of the template under investigation.

### Contents:
* **PCR_template.txt:** Contains the original target sequence in its plasmid form, alongside the specific forward and reverse primers used for the successful PCR amplification.
* **RPA_template.txt:** Contains the linear DNA sequence that resulted from the PCR amplification. This sequence was subsequently used as the template for all downstream RPA reactions.
